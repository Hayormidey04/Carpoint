# CarPoint Website

This is a responsive website for a fictional car dealership called CarPoint. It was created using HTML and CSS and includes a header, hero section, cars section, about section, parts section, blog section, and footer.

## Getting Started

To view the website, simply open the `index.html` file in your web browser.

## Project Structure

* `index.html`: The main HTML file for the website
* `style.css`: The CSS file for styling the website
* `images/`: A folder containing images used on the website

## Features

* Responsive design: The website adapts to different screen sizes, including desktops, tablets, and mobile phones.
* Hero section: A visually appealing hero section with a heading, subheading, and call-to-action button.
* Cars section: A section showcasing the cars available at the dealership, with images and descriptions.
* About section: A section providing information about the company.
* Parts section: A section showcasing the car parts available at the dealership, with images, descriptions, and pricing.
* Blog section: A section featuring blog posts about car-related topics.
* Footer: A footer with copyright information and social media links.
